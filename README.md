# Trade Credit Web App

## Purpose

This is a full stack web application that allows business users from a given company to manage their customers' applications for trade credit (net 30/60/90 days), as well as their ongoing trade credit accounts once an application is approved.

Managers can have a great overview of trade credit applications and accounts by tracking their progress and helping their team stay on track.

Business development reps, salespeople, credit analysts, accounts receivable team can use this web app for day-to-day operations. These include:
    - Adding new prospects
    - Adding new customers
    - Adding a trade credit application for a given prospect/customer
    - Adding files and related data to a trade credit application
    - Approving or rejecting a trade credit application
